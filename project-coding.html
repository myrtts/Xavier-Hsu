About This Website

This website itself is one of my projects! Built with HTML, CSS, and VS Code.

Key Features

  • Multi-language Support: Users can switch between 4 languages (English, Chinese, Japanese, Korean) and the preference is saved locally.
  • Responsive Web Design (RWD): The layout adapts seamlessly to different screen sizes, from mobile phones to desktops.
  • Interactive Animations: Rich CSS animations and transitions are used to create a dynamic and engaging user experience.
  • Robust Contact Form: An elegant contact form with floating labels and a unique animated send button.
  • Clean Codebase: The project is built with clean, well-structured HTML, CSS, and JavaScript, making it easy to maintain and expand.

Technologies Used

  • HTML5
  • CSS3 (Flexbox, Grid, Animations)
  • JavaScript (ES6)
  • Visual Studio Code
